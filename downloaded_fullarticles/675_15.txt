</p><p class="p1">
  <span class="s1">从教室跑出来，下午阳光灿烂明媚。</span>
</p><p class="p1">
  <span class="s1">「喂，你还有课吗，要一起回hall吗。」</span>
</p><p class="p1">
  <span class="s1">金发女孩问。</span>
</p><p class="p1">
  <span class="s1">哈利想，操，世界上怎么会有这种人啊，到底？</span>
</p><p class="p1">
  <span class="s1">结果嘴张开，她回答：「可以啊。」</span>
</p><p class="p1">
  <span class="s1">她们走出学校，一路往hall走。</span>
</p><p class="p1">
  <span class="s1">走到半路哈利才突然想起一个问题：</span>
</p><p class="p1">
  <span class="s1">「喂，你怎么知道我住哪个hall啊？」</span>
</p><p class="p1">
  <span class="s1">金发女孩露出相当古怪的表情。</span>
</p><p class="p1">
  <span class="s1">「继续走你就知道了。」</span>
</p><p class="p1">
  <span class="s1">她没头没脑说。</span>
</p><p class="p1">
  <span class="s1">学校离哈利住的hall也就二十分钟路程。</span>
</p><p class="p1">
  <span class="s1">她们刷卡进入宿舍门禁，一起走入电梯。</span>
</p><p class="p1"> </p><p class="p1">
  <span class="s1">「你几楼啊？」哈利问。</span>
</p><p class="p1">
  <span class="s1">「你自己走你的咯。」金发女孩随便地说。</span>
</p><p class="p1">
  <span class="s1">「傻逼。」哈利评价道，然后按下九楼按键。</span>
</p><p class="p1">
  <span class="s1">hall是男女混住，每层算一个department。长长走廊，昏暗灯光。</span>
</p><p class="p1">
  <span class="s1">宿舍是每人一间房，每两间房共用一个盥洗室。盥洗室把两间房连接起来，从哪边都可以进入。</span>
</p><p class="p1">
  <span class="s1">哈利开学两个月，从未跟和自己的「厕友」打过招呼。她每次进入盥洗室，先把两扇门都反锁好，迅速用完后就回到自己房间。</span>
</p><p class="p1">
  <span class="s1">九楼到了，哈利下电梯，金发女孩也下电梯。</span>
</p><p class="p1">
  <span class="s1">「你是跟踪狂吗？」哈利终于忍不住问道。</span>
</p><p class="p1">
  <span class="s1">「喂，只许你一个人住宿舍吗？」</span>
</p><p class="p1">
  <span class="s1">哈利没话讲，只好往自己房间走。</span>
</p><p class="p1">
  <span class="s1">她的房间在最尽头。</span>
</p><p class="p1">
  <span class="s1">高跟鞋一直跟着她走。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">走到房间门口她才发现，金发女孩停在了隔壁宿舍门口。</span>
</p><p class="p1">
  <span class="s1">「你…… 」</span>
</p><p class="p1">
  <span class="s1">「对。」金发女孩露出一个顽劣笑容，「你就是那种他妈开学两个月，连隔壁宿舍的人都不认识的<strong>傻逼</strong>——我刚开始还以为你是个<strong>书呆子</strong>——」</span>
</p><p class="p1">
  <span class="s1">金发女孩意有所指地指了指哈利脸上那副大大的圆框眼镜。</span>
</p><p class="p1">
  <span class="s1">「不过今天下午我算知道了，就算你是个书呆子，也是个不太正经的那种。最起码是<strong>敢在物理课上自慰</strong>的那种——蛮不错的，我喜欢。」</span>
</p><p class="p1">
  <span class="s1">说完之后，她潇洒地拿着门禁卡在宿舍门上刷了刷，然后进去，「砰」地关上了门。</span>
</p><p class="p1">
  <span class="s1">操，什么人。</span>
</p><p class="p1">
  <span class="s1">哈利也打开自己的宿舍门，「砰」地关上门。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">大概过三秒之后，她打开了通往厕所的门。</span>
</p><p class="p1">
  <span class="s1">几乎同时，对面的那家伙也打开了通往厕所的门。</span>
</p><p class="p1">
  <span class="s1">她们俩站在各自的门旁边，互相对视。</span>
</p><p class="p1">
  <strong>
    <span class="s1">「你好，厕友。」金发女孩言简意赅打招呼。</span>
  </strong>
</p><p class="p1">
  <strong>
    <span class="s1">「你好，傻逼。」哈利言简意赅回应。</span>
  </strong>
</p><p class="p1">
  <span class="s1">「既然都认识了，不如以后这两扇门就不要关了，你来我这边玩吗？」</span>
</p><p class="p1">
  <span class="s1">金发女孩热情邀请。</span>
</p><p class="p1">
  <span class="s1">「绝不可能。」</span>
</p><p class="p1">
  <span class="s1">哈利「砰」地关上了厕所门，然后按下了反锁按钮。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">狭小宿舍里有一张床，一张桌子，一面墙的衣柜和一张可怜的小小长沙发。</span>
</p><p class="p1">
  <span class="s1">哈利把书包随便地扔在地下，然后急急忙忙走到衣柜前。</span>
</p><p class="p1">
  <span class="s1">她的衣柜门虚掩着。</span>
</p><p class="p1">
  <span class="s1">哈利把衣柜门打开，里面右下角有三只抽屉。另一边凌乱地堆了很高的衣物。</span>
</p><p class="p1">
  <span class="s1">她把手伸进去摸索半天，终于摸出几样东西。</span>
</p><p class="p1">
  <strong>
    <span class="s1">一把格洛克17，一把格洛克26，还有两只备用弹夹。</span>
  </strong>
</p><p class="p1">
  <span class="s1">她哗啦一声把衣柜中的最上面的一扇抽屉拉开。里面有一些化妆品的小盒子，还有一只小小的袖珍手机。她把手枪和弹夹放进去，又拿了几件衣服把这些东西盖上。</span>
</p><p class="p1">
  <span class="s1">然后她关上了抽屉。</span>
</p><p class="p1">
  <span class="s1">她站着想了一会儿，然后又拉开了最下面的抽屉，从一堆润滑剂、避孕套盒子和玩具中摸出了一把小小的锁头。</span>
</p><p class="p1">
  <span class="s1">她锁上了第一扇抽屉。</span>
</p><p class="p1">
  <span class="s1">接着，她把衣柜翻的更乱，把衣柜门随便地关一扇门，另一扇门半开着。</span>
</p><p class="p1">
  <span class="s1">就这样吧。她一屁股坐到了床上。</span>
</p><p class="p2"> </p><p class="p2"> </p><p class="p1">
  <span class="s1">通往厕所的门砰砰响起。</span>
</p><p class="p1">
  <span class="s1">哈利不动。</span>
</p><p class="p1">
  <span class="s1">金发疯女孩的声音响起来：「喂，哈利·波特，开门！」</span>
</p><p class="p1">
  <span class="s1">她叹了一口气，认命似的站起来，打开了通往厕所的门。</span>
</p><p class="p1">
  <span class="s1">金发女孩脱了裙子，只穿一条开档丝袜和一条衬衣大大咧咧站在盥洗室日光灯下。</span>
</p><p class="p1">
  <span class="s1">「你要干嘛？」</span>
</p><p class="p1">
  <span class="s1">「我当然是和我的书呆舍友自我介绍。我，叫德拉科·马尔福。」</span>
</p><p class="p1">
  <span class="s1">「你的名字也不像女孩。」</span>
</p><p class="p1">
  <span class="s1">金发女孩出人意料地相当爽快表示了同意：</span>
</p><p class="p1">
  <span class="s1">「是他妈不太像女孩名，我妈起的，他们家所有男孩都用星座命名。」</span>
</p><p class="p1">
  <span class="s1">哈利不由自主地看了一眼那家伙开档丝袜露出的地方：</span>
</p><p class="p1">
  <span class="s1">「你是女的。」</span>
</p><p class="p1">
  <span class="s1">「就是嘛。」德拉科颇为不赞许地说，「也不晓得脑子怎么想的，明明说好是男孩才用星座，结果给我也用了星座名——真是的。」</span>
</p><p class="p1">
  <span class="s1">「德拉科是什么星座？」</span>
</p><p class="p1">
  <span class="s1">「天龙座——喂，我不是来和你科普星星的啊，我是来问你，可不可以借震动棒给我？」</span>
</p><p class="p1">
  <span class="s1">说着，德拉科大大咧咧坐到了长沙发上，腿张开，对哈利露出粉色耻处。</span>
</p><p class="p1">
  <strong>
    <span class="s1">「我还没潮吹呢。」</span>
  </strong>
</p><p class="p1">
  <span class="s1">德拉科认真讲到。</span>
</p><p class="p1">
  <span class="s1">哈利觉得脑袋剧烈地疼痛起来——操，她的厕友真的是个疯子。</span>
