    <p class="p1">
  <span class="s1">一</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">二十四岁后生活开始变得面目模糊，如一个化浓妆在暗暗灯光下做活的五十岁老||鸨。连带面目模糊的是他自己。站在镜子前他时常恍惚。伸左手，镜里人跟他伸左手。伸右手，镜里人跟他伸右手。他感到迷惑。镜中人是谁呢。是他吗。好像是的。因为镜中人同样有绿眼睛和伤疤。那么所有绿眼睛和有伤疤的人都可以是他。“傲罗新秀”可以是他，“救世之星”可以是他，“大难不死的男孩”“女巫周刊最佳微笑”“打败黑魔王的勇士”……</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">都可以是他。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">可他到底在哪里呢。他到底是谁呢。他快忘记自己是什么样了。记得他样子的人死了大半。那些活下来的一一开始自己的生活。全新的生活。人们争先恐后和过去告别，就像在九又四分之三站台的火车上冲送自己上学的父母告别的小孩一样，一样用力挥舞双手，欢呼雀跃。但他的火车误了点。他站在白雾蒙蒙之中，看不见前方后方。他离开不了。他被困其中。汽笛声响起一声又响起一声。都是别人的欢心雀跃，和他无关。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">但幸好年岁渐长让他开始习惯了生理意义上的孤独。在罗恩赫敏买婚戒度蜜月装新房的所有时间里，他学会一个人上班下班买晚饭和看杂志。在有星星和月亮的晚上，他偶尔裹着被子坐在公寓阳台上发呆，感觉到寂寞和隐秘的悸动蛇一般从地上发芽而后缠绕心头。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">有时他想他大概也不应该自怜自艾把自己划为孤独症候的一群里。他有朋友。亲如家人一般的朋友。可以托付性命的朋友。尽管他们最近联络稀少。他喝一口黄油啤酒或伏特加。不知道胃里腾腾烧起来的算什么样的火苗。有欲||望在燃烧。他还是知道这一点的。可是欲||望模糊。像他小时候见到过的寄附在奇洛后脑勺上的那张脸。分不清真正面孔。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">如虫蛰伏。</span>
</p><p class="p2"> </p><p class="p2"> </p><p class="p2"> </p><p class="p1">
  <span class="s1">二</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">二十四岁那一年他还退订了所有的正经报刊和杂志。因为那些纸上总漂浮着和他同名同姓的幽灵。或许也不是幽灵。而是他真的死掉了。他的尸首顺着黑湖漂流，他的幽灵就被钉在黑纸白字和历史书上。十字架钉着，戴着镣铐。有时他在字母和字母之间试图寻找自己的脸，却总是可悲的失败。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">合上杂志他仍然面目模糊。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他大概早就死掉了。人们帮他做好一张面具盖在他尸体上。他死掉了。在面具之后。身体摇荡。面目模糊。他死掉了。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他死掉了。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">但他还能去上班，去聚会，去和朋友们开无伤大雅的玩笑。但他死掉了。他不能撒尿，不能排泄，不能勃||起。死是一种漫长的凌迟，从他十七岁开始，或者更早，把他身上所有不好的、坏的、邪恶的棱角一点一点刮掉。不疼。但让人怅惘。他已经记不起应该怎么狠狠地啐一口小报记者、或者色迷迷地对着性感女郎吹口哨了。虽然他以前也没有这样做过。他不该这样做。他是哈利·波特。他十全十美。于是每一天醒来他都发现自己比昨天死掉一点。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他高高在上他善良温和宽容坚毅他保护无辜者他公平公正他魁地奇永远抓得到金飞贼……</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">他死掉了。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">有一次他在一场扯淡见面会上。一个小巫师从人群里挤出来递给他一只巧克力蛙。他对她说谢谢，又拆开巧克力。青蛙蹦到了他眼镜上。他把它扯下来塞到嘴里。他这才看见这只巧克力蛙带着的是邓布利多的卡片。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">阿不思·邓布利多 曾任霍格沃茨校长</span>
</p><p class="p1">
  <span class="s1">被公认为当代最伟大的巫师</span>
</p><p class="p1">
  <span class="s1">邓布利多广为人知的贡献包括：</span>
</p><p class="p1">
  <span class="s1">一九四五年击败黑巫师格林德沃，</span>
</p><p class="p1">
  <span class="s1">发现龙血的十二种用途，</span>
</p><p class="p1">
  <span class="s1">与合作伙伴尼克·勒梅在炼金术方面</span>
</p><p class="p1">
  <span class="s1">卓有成效，邓布利多教授爱好室内乐</span>
</p><p class="p1">
  <span class="s1">及十柱滚木球戏。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他的老校长，长长白色胡子和半月形眼镜，蓝色眼珠冲他闪烁。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“你是什么时候死掉的？”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他问。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“噢，在死亡的很久之前，我的孩子。”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">老校长说。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">于是他明白他注定死去。他已经死去。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">在他真正死亡之前。他死去。</span>
</p><p class="p2"> </p><p class="p2"> </p><p class="p2"> </p><p class="p1">
  <span class="s1">三</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">重新遇到德拉科·马尔福也就是在他二十四岁那一年。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">那段时间他接了一个追踪非法交易黑魔法物品的任务，时常在翻倒巷蹲守。每隔一个小时他往随身带的杯子里扔一根不同头发，喝一口冷冰冰的复方汤剂，保持陌生人容貌混迹在这里。也是那段时间他感觉到有人在跟踪他。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">二十四年生活予他敏锐的直觉，附赠伤疤和暴躁。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他呼气，在冬日空气里制造一场小小的雾。翻倒巷的石板路上一群蜘蛛排着队在寒霜覆盖的地面上行走。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他不打算把这件事告诉任何人。他想得到这一切的后果。罗恩会迷茫地皱起眉头：跟踪你？有人？我说，不应该呀，是吧？谁会跟踪你呢？神秘人……伏地魔都死了，对吧？而赫敏会大惊小怪地要求他告诉金斯莱告诉他所有的傲罗同事告诉韦斯莱夫人并且改用飞路粉上班，她还会逼他去心理咨询逼他休假逼他谈女朋友总之逼他做一切大惊小怪的事。他爱赫敏。当然。他爱罗恩。毫无疑问。他爱他们，比谁都爱。但他烦他们。就像青春期的子女烦唠叨的母亲与木讷的父亲一样烦他们。他决定把被跟踪的秘密锁起来。谁都不告诉。他要去自己抓到那个人，抓到他。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">就在他下定决心找到那个跟踪者的第二天，他碰到了马尔福。在翻倒巷的博金博克商店旁边。他总是能碰到他的老地方。马尔福穿着一件黑色的斗篷，混在一伙他早有怀疑的巫师小团体中间。哈利一眼就发现了他。于是他躲在街角。等到马尔福淡金色的头发刚刚露出一个角，他就敏捷地抓住了他的袖子把他拖到了一条小巷中。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“波特？”马尔福惊讶地挑了挑眉毛。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他哼了一声作为回应。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“你在这儿干嘛？”马尔福迅速调整出一副傲慢的神色。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“这话我应该问你。”他抽出魔杖在手指间慢慢转。“你交朋友的品味变了？现在想和倒卖黑魔法用品的人插一脚了？”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">马尔福翻了个白眼，懒洋洋交叉双臂。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“你随便讲话我告你诽谤。”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">哈利再次对马尔福冷哼了一声。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“跟我回魔法部，录个口供再走。”</span>
</p><p class="p2"> </p><p class="p2"> </p><p class="p2"> </p><p class="p1">
  <span class="s1">四</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">二十四岁那一年他还和金妮·韦斯莱分了手。分手很低调。他们在陋居的阁楼上接了个轻的不像话的吻。金妮身上还带着那种淡淡的花香——他曾经在斯拉格霍恩的魔药课上闻到过。但是这味道已经不再让他怦然心动。他抚摸金妮火红色的头发，想念它们曾经在他指头上烙下的炙痛痕迹。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">那仿佛已经是很遥远、很遥远以前的事情。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">他想起很久之前他曾做过的梦，罗恩掀开巨怪棒打傻巴拿巴的挂毯，看到他和金妮在接吻。那时候他才上六年级呢。金妮和迪安还没有正式分手。那时候的时间回想起来是如此漫长，一切都在记忆里被拉扯成慢镜头，一切都被笼罩上了难以除去的灰尘。他住在陋居，他和金妮坐在地板上。他们聊天，微笑。冬天还没有降临。金妮卧室的窗外，秋天最繁盛的金色阳光，像甜美果实一样结得硕果累累。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">你还记得禁林吗？我们曾经约会去过的地方。我记得。还有黑湖。七楼的空教室。皮皮鬼从我们头顶飞过。我都记得。他们慢悠悠地讲话，友好如一对甜蜜朋友。哈利下巴支在膝盖上。关于霍格沃茨、关于一切还没有开始前、无忧无虑（最多为作业啊恋爱啊忧虑）的少年时光重新温柔地涨上他心头。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">禁林。鹰头马身有翼兽。海格。他想起来。许许多多过去的事情，一件又一件，闪着金光，在他脑海里发出温热的熨贴温度。紧接着，他想起了马尔福。那个十足十的混蛋。那混蛋有一头在阳光里会闪光的金发，会在魁地奇比赛里用扫帚狠狠地撞他，一天三次地来到格兰芬多里和他找茬。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">他想的出了神。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“还不下班吗？”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">傲罗办公室的实习生敲了敲他紧闭的办公室门。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他被吓了一跳，手里的口供也掉下来。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">是德拉科·马尔福的口供。关于他为什么会出现在博金博克商店。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“这就走。”他回答。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他拎起椅背上的外套。裹好羊毛围巾。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">走到办公室门口时他看到镜子里有一个褐色头发、满脸雀斑的高个子男人。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">原来复方汤剂药效还没有过。他想。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">紧接着他的动作僵硬起来。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">复方汤剂药效还没过——可这样马尔福都能认出他。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">就好像十七岁的那个冬天——他鼻青脸肿，被抓到马尔福庄园里，但马尔福还是一眼认出了他。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">那家伙在认出他这方面还真是天赋异禀。</span>
</p><p class="p2"> </p><p class="p2"> </p><p class="p2"> </p><p class="p1">
  <span class="s1">五</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">二十四岁的一个冬夜他带着三个实习生去抓捕那窝倒卖黑魔法物品的小贩。在被秘鲁烟雾弹突然袭击后，这次普通的抓捕任务变成了一次出乎意料的激烈打斗。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">普通小贩不会这么拼了命的和傲罗打架。这伙人很有可能只是打着卖黑魔法物品的名义，聚在一起商量其他更危险的事。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">他必须把他们抓住。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">他并不畏惧这种场合。相反，他享受这个。在躲闪和魔杖挥舞间他恍惚以为自己是一种兽类，如同他的教父，有湿润的鼻和灵巧的嗅觉。他扔出一个个咒语，剑客一般挥洒自如，在狭窄街道上追赶逃窜的嫌犯。就在他几乎要抓住其中一个的时候，他听到他的实习生惨叫了一声。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">他转头，前面的嫌犯迅速抓着他的腿把他放倒，并且恶狠狠地发射了一个割裂咒。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">血瞬间从他的额头上汩汩地流出来。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">先是黑。然后是白。最后是一片茫茫然的絮状背景。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他躺在地上，想，真冷啊。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">他果然还不是兽类。他永远不能像大黑狗一样安然趴在雪地上，却一点都不觉得冷，浑身还散发着热气。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">他需要温暖。</span>
</p><p class="p2"> </p><p class="p2"> </p><p class="p2"> </p><p class="p1">
  <span class="s1">六</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他醒过来的时候是在圣芒戈。他太熟悉这里的味道和声音了——自从当了傲罗，他常常来这里来探望队友，不过很少自己住进来就是了。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">治疗师们在高声嘱咐着病人药剂药量。护士们匆匆地给每个人分发精神滋补剂。病人们诶哟诶哟地叫唤着。这一切织成一匹宽柔的丝卷，笼罩在他病房门外。他的病房很安静。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">他从床上坐起来。发现视线狭窄的可怜。于是他伸手去摸自己的脸。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“你最好别乱动。”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">有人说。他停下手上动作。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“你的左眼受了伤，刚刚给你扎好了绷带。如果你弄坏了，就当个独眼龙去吧。”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他皱着眉看马尔福穿着白色长袍拿着病历本朝他走过来。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“你在这儿干嘛？”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他问。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“杀你。”马尔福冷冰冰地说，从床头柜上拿过一只大杯子。“喝了它。”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“什么？”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“生骨灵。”马尔福看着他惊恐的脸假惺惺地微笑，“就是那个很难喝很恶心的药，没错。”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“我觉得我挺好的，用不着这个。”他抗议。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“那你就抬抬你那条小细腿看看。”马尔福在他床沿边上坐下来。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他抬了抬他的腿。一阵剧烈的痛感从膝踝处升上来。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“好吧。好吧。”他接过马尔福手中的杯子，屏住气一口喝了下去。“操，我的嗓子要烧掉了，给我点水！快点！”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“没有。”马尔福高兴地说。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“靠——我魔杖呢！”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“病房里不许用魔法。”马尔福咧着嘴说。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“我要杀了你。混账。”他嘟嘟哝哝地说。“你穿着治疗师的袍子干嘛？偷衣服玩？”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“你只瞎了一只眼，波特。你另一只眼是干嘛用的？”马尔福冷笑，“你的蠢脑袋被撞了一下就不认识字了吗？”</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">他指着衣服口袋边的“见习治疗师 德拉科·马尔福”。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">“如果我是你，我就会对治疗师表示出一些尊重，毕竟是他抢救了你那颗硕大无比的头颅。”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“我当然看到了你的胸牌，否则我不会从一个前食死徒的手里拿过药——真是十二万分地感谢你。”哈利挖苦。但是话音未落，他突然被自己逗笑了。他也不知道为什么。总之他的心情变得十分开朗，生机勃勃。带着一种他很久没有体会到的那种、想和别人争强斗胜的蠢蠢欲动。</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">现在马尔福在他眼前，整个人比上学的时候还招人讨厌：</span>
</p><p class="p2"> </p><p class="p1">
  <span class="s1">还有比和马尔福斗嘴更适合的康复运动吗？</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">没有。</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">在他来得及提出下一个挑衅话题前，马尔福张了嘴：</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">“你差不多了要不要出门看一下？大概有十家报纸正守在门口要采访你。你可怜兮兮的忠诚粉丝团排队排到了圣芒戈门口，都在哭啼啼地为你写赞美诗：伟大的救世主，为了自己的实习生和魔法世界的安定牺牲自己的个人安危……”</span>
</p><p class="p1">
  <span class="s1"> </span>
</p><p class="p1">
  <span class="s1">他的脸迅速地阴沉了下去。</span>
</p><p class="p2"> </p><p class="p2"> </p><p class="p3">
  <span class="s1">七</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">人群惯于谄媚和幻想。</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">他被一团团记者包围住。镁光灯扑哧扑哧地闪。他知道明天的预言家日报上会怎么写。忠于职守勇敢无畏冲锋陷阵。他们会无视事实拼命地夸赞他，就像他们曾经无视事实拼命地贬低他一样。他成为了一只木偶，一尊神像，专供安全感匮乏的人顶礼膜拜。他呆呆地坐在那里，本能地回答那些无聊的提问。</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">没人爱他。人人都爱他。没人关心他。人人都关心他。</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">……去他妈的吧去他妈的吧。</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">他感觉怒气突然从胸腔里升起，窜上他脑袋几乎炸裂他的颅骨。他要撒尿，他要排泄，他要勃|起，他要找个人破处然后好好地做爱。他要老，他要死，他要皱纹爬满脸然后流口水得健忘症，变成一个一无是处的糟老头。</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">他在众目睽睽下用幻想糟践自己，得到自残的快感。</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">他终于把目光放到远处，思考若踢翻眼前的桌子，推开拥挤的、散发着臭气的人群，跑出门去，一路跑下去会有什么后果。他往远处看，却看到了马尔福带着抹冷笑靠在病房门口，目光像一柄看不起人的利剑。</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">穿喉而过。</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">他记忆中的马尔福很少有这样凌厉的时刻。他熟悉的马尔福胆小、怕事、摇摆不定、欺软怕硬，是个只会喊爸爸的奶娃娃。但现在的马尔福看起来如一个真真正正的陌生人，有一头灿灿的金色头发，有他不得不承认的、俊俏面容。</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">还有一副冷情的眉眼。</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">欲望撩开了它的面罩。奇洛摘下了头巾。邪恶和幻象开始如云雾遮蔽他的双眼。德拉科·马尔福是个完美的工具。他想。如果，他是说如果。如果他和他厮混在一起会怎么样？马尔福的无与伦比之处在于，他是个毋庸置疑的混蛋。十恶不赦的前食死徒。十足的恶鬼和懦夫。</span>
</p><p class="p4"> </p><p class="p3">
  <span class="s1">——还是个男人。</span>
</p><p class="p4"> </p><p class="p3">
  <span class="s1">从哪个方面来说，他都不应该和马尔福扯上关系。</span>
</p><p class="p3">
  <span class="s1"> </span>
</p><p class="p3">
  <span class="s1">可这正是他想要的。</span>
</p><p class="p4"> </p><p class="p3">
