</p><p> </p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“下午好，医生。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思•邓布利多第一次见到盖勒特•格林德沃是在一个充满阳光的下午。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">他从医学院刚刚毕业，这是他第一次离开学校去接受会诊。格林德沃的家就在柏林最负盛名的一条街上，邓布利多背着医药箱走到大门口的时候竟然还有些紧张。他头顶上戴着自己小妹妹阿利安娜织成的帽子，蓝色的羽绒外套和他的赤褐色的头发形成了鲜明的对比。他没带手套，所以手指冻的有些红肿了，敲门的时候那里还会习惯性的蜷缩了一下。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">他没有等很久，开门的是一位金发的女士，她漂亮的湖蓝色眼睛竟和阿不思的有点相似。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">进了门阿不思知道那是他今天会诊对象的母亲——格林德沃夫人。她是个看上去很干练的女士。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“我很担心他的猩红热。因为我们全家下周还打算去美国……”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思跟着格林德沃夫人一层一层走上楼梯，“毕竟目前这个形势，谁都看得出来。”似乎是看阿不思比较好说话，格林德沃夫人的话匣子一下子全都打开了似的。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">是个难得的清醒人。阿不思刚刚想下个结论，他们便来到了盖勒特的房间。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">映入眼帘的卧室布置绝对属于一个富裕家庭的孩子。窗帘厚重的遮盖住玻璃，墙上是电影明星和球星的海报，打字机放在桌子的一角，隐隐拉出的抽屉可见一排排闪亮的奖状勋章。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">盖勒特闻声抬起头来，虽然他还不能大幅度的运动。他有一双和母亲一样的眼睛，此刻正努力的睁开看着阿不思。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">但浑身的红疹和强烈的高烧使他昏沉，阿不思赶紧掏出听诊器，半跪在床边仔细查看男孩的症状。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思小心翼翼地掀开男孩的衣衫，“乳头肿大……喉咙也微微充血，舌头上有白苔。我需要给他测试一下体温。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“他还好吗？”格林德沃夫人语气担忧地问道。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“交给我。”阿不思沉着地吸了一口气，“他已经过了前驱期，还好不严重。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“那么感谢您，医生。”格林德沃夫人作势要走出房间，“今夜您就在家里用餐吧。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">看着格林德沃夫人出去了，阿不思才感到自如些。他一直很不适应有人看着他会诊，在医学院的时候就是这样。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">男孩的症状不算严重，但他们一家人想要去美国的计划应该是要推后了。因为格林德沃家的酬劳的确丰厚，阿不思陪伴了这位少年整整一周。包括为他物理降温，吃退热剂和使用青霉素。总之在几天的治疗下，男孩的病终于见了起色。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思不擅长与人交流，所以除了吃饭时间他也很喜欢在盖勒特的房间里坐着。男孩的屋子里有一台黑白电视，里面会演一些老电影和新闻。阿不思还会打开收音机听着歌，傍晚时分把窗户打开一角，给房间透透气，看着橘红色或紫红色的夕阳悄悄退去。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">那是一段温柔沉寂的时光，外面战火连天，室内温暖如春。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思穿着白色毛衣在房间里昏昏欲睡，昨天他照顾盖勒特几乎一夜没有休息。他把自己缩在盖勒特房间的沙发里，虽然格林德沃夫人已经为他准备好了客房。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思闭上眼睛的时候窗外的阳光还烘着他的脸，热乎乎的。等到太阳都要落山了，一丝丝寒意爬上阳台溜了进来。阿不思被这冷风给弄醒了，他迷糊着走到窗户前面，伸出手将那道打开的缝隙闭合上。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">这时有东西拽住了他的衣角，阿不思低头看去，是一只骨节分明的手。上面的红疹已经消退了大半，指尖微微耸动，似乎带着某种依恋——也有可能是他臆想出来的。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“下午好，医生。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">盖勒特完全清醒过来了。阿不思勾起嘴角，“你好，盖勒特。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">男孩从床上坐起来，然后直接下了床。精壮的胸膛就这样露了出来，盖勒特像是没有在意屋子里还有个人似的，他大刺刺的脱掉裤子，还有内裤。阿不思觉得自己的眼皮猛烈的抖了一下，他听到盖勒特抱怨起自己的病来，害他只能躺在床上，现在他要去洗个澡。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“医生就留在这吧。”在阿不思刚想出去向格林德沃夫人汇报的时候，盖勒特如是说。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">楼下传来汽车轰鸣的声响，盖勒特向大门处瞥了一眼，门卫们正将栅栏打开以便那黑色的驾驶工具可以顺利的出去。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“或许他们今天要很晚才能回来。”盖勒特的语气里泄露出一丝轻蔑，“看来我们今天没饭吃了。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思听着浴室里“哗哗”的流水声有些惴惴不安。他决定下楼去看看，最好是能在厨房里找些食材。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">盖勒特从浴室出来便闻到了饭香，他顺着嗅觉一路走到客厅，发现平时自家用餐的小桌子里多了些煎饼和培根。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">而厨房里忙忙碌碌的身影，自然是对他无微不至的邓布利多医生。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“你的服务太周到了。”盖勒特对此感到赞叹，“这本来应该是我们家保姆做的事情，不过今天是周日，她正好放假。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“你们家有保姆？”阿不思皱紧了眉头。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“是啊。”盖勒特将煎饼放在嘴里，味道真是好极了。“你没有吃过她做的饭吗？”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思将三明治放在桌子上，犹豫之际还是没有把他这一周都在吃格林德沃夫人做的饭的事情说出来。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">咖啡机里还在轰隆隆的煮着摩卡，二人各坐在桌子的一角，阿不思将三明治塞进嘴里咬了一口，颇为不好意思的开口道，“其实我打开电视机，还看了你书架上的书，呃，收音机也听了一些。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“我知道。”盖勒特笑道，“虽然我意识不清醒，但我知道有个人一直陪着我。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“啊，哦。”阿不思一下子不知道说些什么了。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“你很认真，”盖勒特将身体向前倾斜了一点，“一直都是这样吗？”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“其实……不。”阿不思转过头去，想把脑袋塞进衣领口袋里。“你是我毕业后的第一个病人。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“原来是这样……”盖勒特喃喃点头，“医生多大岁数？”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“二十四。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“医生大我九岁。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“是，是啊。”阿不思看着男孩身上只堪堪围着条浴巾，脸颊突然像是烧起来一样。他像是想起来什么似的走到了咖啡机旁边，拿出橱柜上的米色瓷杯，“你要加糖吗？还是牛奶……？”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">有人贴近他了，还有随之而来粗重的喘息。阿不思的喉咙里突然就想要发出一阵声响，或许是声呻吟。他又怎么能注意不到男孩看他的眼神——从他清醒过来的时候便存在了。</span>
</p><p> </p><p class="ql-long-16139928">
  <em class="ql-author-16139928">搅动的欲望。</em>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">哦，上帝，那个男孩明明才十五岁啊。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">他的小腿肚子上骤然增加了些重量，盖勒特很高，几乎能把他整个包裹在怀里。阿不思为他不大的年龄感到振奋，他难以想象盖勒特再长大些会是什么样子。阿不思觉得自己要做出一点什么反应了，至少别这样僵着一动不动。现在盖勒特亲到他的耳廓了，那里的绒毛正因为男孩的动作而瑟缩发抖。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“你有没有觉得有什么味道？”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思觉得自己简直是个废物，他马上就要和一个年轻的男孩做爱了，却如此煞风景的吐出这么一句话。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“味道？”盖勒特停了下来，他的手还搭在阿不思的腰间。他们对视了一眼，食物和咖啡的香气开始散去，一股几乎是腥臭的气味顺着厨房的盥洗盆直冲冲的钻进两人的鼻孔。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">shit！</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">有点像鱼或者是什么东西腐烂了的味道，盖勒特和阿不思意识到事情有些不对路。他们彼此分开了，这令阿不思为刚才差点发生的事情感到一阵转瞬而来的羞赧。但他来不及想那么多了，他们一路顺着气味向上，走到了阁楼的仓库。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“这里？”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">气味愈发浓烈了起来，盖勒特用手捏住了鼻子，而阁楼的仓库钥匙只有他家的保姆格里姆森太太和他妈妈才有。盖勒特用脚使劲踹了几下，门随着撞击砰砰作响。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">撞击了大概十几下左右，仓库的门终于被打开了。盖勒特拽住阿不思的手，刚进去便被眼前的景象给震惊了。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">仓库里堆满了废弃的纸盒还有布满灰尘的废旧家具，煤油灯将整个逼仄的小屋子照的灰亮，蝇虫像是在这里扎了窝，一股脑地围绕在躺在地面中间的女人。她额头上的血已经干涸，皮肤松弛，恶臭味就像是海浪似的扑向阿不思二人。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“是格里姆森太太……”盖勒特的眼里全是不可置信。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“她死了。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">这是一个显而易见的事实，而且这位格里姆森太太死了至少有三天。阿不思迅速的反应过来，“我们得将她转移走，不然整间屋子都会充满尸臭味的。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">盖勒特拿起一把凳子将仓库棚顶的楼梯放了下来，他“嘎吱嘎吱”地走上去，又“嘎吱嘎吱”地走下来，然后冲着阿不思挥了挥手：“我们把她拖到楼顶上去。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">这似乎是个最好的方法。阿不思先牵起格里姆森太太的胳膊，将她拖到楼梯上去，然后盖勒特抓住她的腿，减少阿不思所承受的重量。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">死人往往比活人要重，何况格里姆森太太生前又是个丰腴的女士。他们将她转移到楼顶时便有些气喘吁吁了，在他们放下格里姆森太太的时候，她那湖蓝色的貂皮大衣里面还跑出了一只老鼠，直接吓的阿不思一个踉跄差点倒下。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“所以……”在他们二人沉默了片刻后，阿不思率先开口，“你家里发生了凶杀案。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">刚刚朝仓库走去的时候盖勒特随手套了件外套，他掏出里面的烟，也许是他的，也许是他爸爸的。火光划过空气，盖勒特的动作老练极了，阿不思不禁开始思索他的烟龄有多久了。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“我好像还没有跟你说过……”盖勒特似乎在斟酌如何用词，“我爸爸不是什么好人。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">这几乎是一句暗示。阿不思不知道该如何回应，他只是机械地点了点头，等到盖勒特继续说下去。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“可能格里姆森太太发现了什么不对劲的东西。”盖勒特重新将视线放回死去的格里姆森身上，应该是想到了他的父亲，他的目光里的厌恶一闪而过。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“他杀了她。”阿不思总结道。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“没有别的可能。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">天色开始变暗，太阳从西边落下。阿不思打了一个冷战，他把手插进裤兜里然后说道：“我想我该走了，而且你的病也好了。”</span>

</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">她说的是，“凤凰福克斯。”</span>
</p><p> </p><p> </p><p class="ql-long-16139928" align="center">
  <strong class="ql-author-16139928">1937</strong>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“从今天开始，你的代号就叫凤凰。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思十八岁的时候正式成为了一名间谍，接替他的爸爸铂西瓦尔•邓布利多。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">高个的长官喜欢用手抚摸他的肩头，用沙哑的音色跟他说话。他曾经多次见到自己的父亲与这个男人来往，他喜欢和弟弟阿不福思一起趴在窗户口看着他们一起钻进一辆车离开，而母亲坎德拉总是对此不安。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">这份不安终于在铂西瓦尔被捕入狱之后结束了。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">作为一名间谍，暴露身份便等于职业生涯的陨落，何况他的父亲还为此锒铛入狱——这堪称耻辱。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思没见到父亲的最后一面，甚至连父亲的死讯还是加入了军情六处才知道的。年轻的长官需要一个邓布利多家族的人，而身为长子，他被选上了。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思几乎没有抵抗，即使他凭借着优异的成绩考入医学院在众人看来是多么的前程光明。他知道自己内心那份蠢蠢欲动的阴暗面，他想和父亲一样，但又要比他强。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">二十五岁他完成了自己最后一次任务，至少他是这样想的。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">他把铂西瓦尔欠的都还清了。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">三十一岁的时候阿不思做了两件自己认为有意义的事，回到军情六处和寄给盖勒特第一封信。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“欢迎你回来，阿不思。”长官站在他面前，他已经老了一些，但眼睛还如同当年一般犀利。“这次是什么促使你重新找到我？”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">是一想到和那个男人做同一种职业会让他兴奋？还是这样说不定会离那个男人更近一点，阿不思不记得自己具体是如何回答的了，他只知道身体内重新涌起的那种鲜活的感觉。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">也许他和盖勒特就像悬崖两端的峭壁，天生便适合互相对望。</span>
</p><p> </p><p class="ql-long-16139928" align="center">
  <strong class="ql-author-16139928">1953</strong>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“你还会回我的信吗？”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">船舱内，红发的男人看着天花板上虫子爬过的路线。他看似随意的问道，如果忽略他被扯的歪歪扭扭的领带和空气中的浓精味，一切都看上去很正常。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">本来坐在里面稳稳拿着文件的他没想到和盖勒特的第三次见面如此猝不及防。他想过什么时候把福克斯的称号不经意的透露给西德，又或许他出任务的时候制造一次“偶遇”，又或许他就如同自己信里说的那样，“在伦敦当着大学老师”。那么普通的人生，永远不会和如今的西德第一情报官有交集。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“你就那么喜欢做这种事吗？和十年前一样，被人利用、被数不清的危险所伤害？”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">是的。他没等自己说话，便用自己的嘴代替了回答。盖勒特几乎是肉眼可见的速度硬起来的，就在阿不思将头停在他胯前的时候。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">列车上的那次不过是含着玩玩，这次的才是“成人游戏”。频繁的深喉使阿不思的眼角发红，他被迫抓紧男人的衣摆保持平衡，快速的进出使他发出“吭哧”“吭哧”的声响；吞精的时候他把眼睛闭的死死的，像是享受又像是哭泣。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“你写的都是假的，整整三年。”格林德沃将自己的脸隐藏在阴影处，香烟的味道钻到阿不思的鼻子里。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“是。”他无言反驳。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“为什么骗我？”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思不说话，盖勒特叹息了一声，他将烟蒂踩在脚下，双手捧起阿不思的脸，就好像三年前那样。</span>
</p><p> </p><p class="ql-long-16139928">
  <em class="ql-author-16139928">你该好好过正常人的生活。</em>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">他几乎就这样说了，但盖勒特只是将头靠在阿不思的头上，“小心点。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“我会的。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">男人抱住了他，两人静静的感受着对方的呼吸。“我该走了。”盖勒特松开了阿不思，“谢谢你的情报，福克斯先生。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“那是代号，我现在叫布莱恩•伍德。”阿不思笑了出来。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“好的，伍德先生。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">门打开那一刻，盖勒特回头深深地看了阿不思一眼，“我一直留着那把枪。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">这句话就仿佛无声的告白，紧紧的抓住了二人。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“再见，盖尔。”年长的爱人微笑着。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">只有在阿不思面前，他才能做回盖勒特•格林德沃，那是埋藏在十年前的小秘密。海浪不断奔跑至岸边，货船慢慢地在盖勒特的视线中驶离了港口。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">这次，他们都不会仓皇离去了。</span>
</p><p> </p><p> </p><p class="ql-long-16139928" align="center">
  <strong class="ql-author-16139928">1961</strong>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">在1961年的夏季，有两件事值得人们注意。全长155公里的柏林墙始建，铁丝网及瞭望台开始成为德国人的噩梦；斯塔西（西德情报机构）有“黑魔王”之称的间谍人员帕西瓦尔•格雷夫斯的失踪。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">年初他化名为“西里尔•伊万诺夫”进入到苏联境内，正式离开波恩，成为了一个被派遣在异国的间谍。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">在蛰伏了将近半年之后，8月他突然被斯塔西召回。路线、过程、交接人员都是安排好的，除了——</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“帕西瓦尔•格雷夫斯，抓到你了。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">站在机场的安检口，他看到了警员脸上闪过的一抹狐狸般的微笑。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">格林德沃被押解至机场的收容室，近十名克格勃（苏联情报机构）的人员对他进行了搜身和看管。全程他都保持沉默且没有抵抗。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">没有人员对他进行搭救。格林德沃意识到了自己所处的境地。</span>
</p><p> </p><p class="ql-long-16139928">
  <strong class="ql-author-16139928">
    <em>他被国家背叛了。</em>
  </strong>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">不过就在那天的凌晨时分，前往莫斯科的飞机中途坠毁，苏联警察赶到之时便只有几具烧焦的尸体和飞机残骸。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">克格勃在半个月后对外公布王牌间谍帕西瓦尔•格雷夫斯在飞机事故中死亡。原因是他们鉴定了其中一具尸体的腰间别有格雷夫斯常年携带的手枪。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“牺牲一个间谍所带来的收益往往出乎你的意料。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">但如果那个间谍从来都没有离开过呢？</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">所有的间谍都有一个共识——不做人群中最显眼的那个，保持警惕，学会隐藏，在风声鹤唳之前准备好一切。</span>
</p><p> </p><p class="ql-long-16139928">
  <strong class="ql-author-16139928">甚至是，准备好逃跑。</strong>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">◇</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“你的任务是什么？”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思还记得1943年在丛林中盖勒特问他的话，“隐藏——”他当时是这样说的，“——观察”。“作为组织的眼睛，引导组织的利剑。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“每走一步，都要回头看一步。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">当他还穿着水手服，在湖边肆意玩耍的年纪时铂西瓦尔曾对他说过，“你只是审视自己走的稳不稳，而不是后不后悔。你永远不要后悔自己做过的事，我的儿子。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">他朝自己开了一枪，没有犹豫。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“伍德先生！”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">不知情的人们朝他涌来，鲜血如同泡沫一般冒出他的口腔。在迷迷糊糊倒下之际，一双异色的瞳孔出现在他的脑海里。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">格林德沃在伦敦，这是他在克格勃宣布他死亡之后发现的事情。起因是他住所的猫开始逐渐减少进食，起初他以为是被人跟踪了，后来他才发现门锁里藏着的一个个微型胶卷，里面全部都是利于他的情报。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">有人喂他的猫，有人在暗中帮他。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">至少对方表现出来了善意，但阿不思仍不肯放松警惕。而伦敦也开始渐渐多了一些苏联人和德国人，据军情六处分析，他们是来找帕西瓦尔•格雷夫斯的。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">格雷夫斯在伦敦。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">几乎同时，阿不思意识到那个所谓的“好心人”是谁。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">但盖勒特从未给他写过一封信，约他见过一次面。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">一个知晓众多情报却还活跃于世的人自然是斯塔西和克格勃争相抢夺的对象。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">苏联想要盖勒特归顺于他们，所以伪造了他的死亡来迷惑西德；西德却也猜到了那场飞机失事背后的猫腻，和为什么苏联延迟了那么久才公开的“真相”，恨不得将盖勒特杀之后快。</span>
</p><p> </p><p class="ql-long-16139928">
  <em class="ql-author-16139928">现在，他变成了盖勒特的眼睛，盖勒特的利剑。</em>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">阿不思就这样改变了自己的定位，或许一开始，他们就如同两个抱着浮木的人，飘到那方便算那方。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">他们真正在乎的，早就不是什么所谓的正义与和平。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">在送往医院的路上，阿不思陷入了沉沉的梦境。他四十四岁了，自诩经历过许多事情；有杀戮，有死亡。但最后他脑中想到的，还是年少时父亲离开的背影和盖勒特短暂却又炙热的怀抱。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">第二天清晨，阿不思•邓布利多，军情六处现任情报间谍，也被称为“凤凰福克斯”的他安静地躺在病床上。一切都没有变化，除了那件沾血的大衣的袖口上，少了枚纽扣。</span>
</p><p> </p><p> </p><p class="ql-long-16139928" align="center">
  <strong class="ql-author-16139928">1989</strong>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“你所拨打的用户正忙，请稍后再拨。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">当阿不思第三次听到冰冷的机械女声时他果断挂了电话，这次他没有继续等待，而是前往厨房把他早上未完成的松饼放在烤箱里，过不了一会就会有香喷喷的气味充斥在厨房里。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">詹姆•波特，一位戴着圆眼睛的年轻记者在此时敲响了阿不思的家门。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“先生……”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">詹姆挂着笑意看向为他开门的老人，他微微露出了错愕的表情，他没想到这位兰登先生的长相是如此的和善。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">他走进这栋小小的二层楼里，里面被布置的十分整洁，除了壁炉上堆满的甜品盒子。詹姆再次确认了此次他要采访的对象，现年七十一岁的前军情六处间谍，菲利普•兰登。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">在他印象里，间谍一类的角色应该是诡诈的，至少给人的感觉会令人不舒服。但对面这位留着花白胡子的老人却给他了一种学校教授的气场。</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“请坐。”走进客厅后阿不思递上了一杯热茶，“我准备了松饼，只不过它还在加热。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“您客气了。”詹姆赶忙摆了摆手，“非常感谢您这次联络我们的报社，对于您想透露的关于冷战的一些消息我们产生了极大的兴趣。”</span>
</p><p> </p><p class="ql-long-16139928">
  <span class="ql-author-16139928">“是的……我的确了解一些事情。”</span>
