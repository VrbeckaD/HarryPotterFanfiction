    <h3 class="landmark heading" id="work">Chapter Text</h3>
    <p>  “哈莉！”<br/>
　　<br/>
　　佩妮姨妈尖利的叫喊声在门外响起，随之而起的是一阵激烈的拍打门的声音。<br/>
　　<br/>
　　哈莉波特赶紧从床上爬起来，今天是达力表哥的生日，她要负责做饭、洗干净并熨烫好他的衣服以及姨妈和姨父还有达力表哥吩咐的任何事。<br/>
　　<br/>
　　任何事。<br/>
　　<br/>
　　哈莉啐了一声，穿好衣服，戴好自己的破烂眼镜，用一根细绳绑好乱糟糟的头发——要是被姨妈发现有头发掉进亲亲达力表哥的熏肉里，那可有她好受的。<br/>
　　<br/>
　　推开储物间的门，匆匆洗漱完毕，哈莉来到厨房，给达力煎蛋和熏咸肉。<br/>
　　<br/>
　　一阵阵香味涌进哈莉的鼻子里，但她得忍着饥饿把这些送给正在狼吞虎咽的达力，一盘又一盘，吃得又快又快，达力的食量永远是那么惊人。<br/>
　　<br/>
　　等达力吃完，她把餐桌收拾干净，把碗洗完之后，才有时间坐下来品尝她的早餐：一小盘熏咸肉，还有一只煎蛋。<br/>
　　<br/>
　　这是达力表哥实在吃不下去，留给她的。<br/>
　　<br/>
　　等一会儿她就要像往年的生日一样，在佩妮姨妈全家和达力的朋友皮尔出去玩的时候，待在费格太太的家里了，她这么想着——忽然得到消息，费格太太病了，她得跟他们一起去动物园。<br/>
　　<br/>
　　没过一会儿，哈莉就跟皮尔和达力坐在汽车后座，达力一个劲地向皮尔炫耀“哈莉是我的小女仆”之类，告诉皮尔每天哈莉都得给他当牛做马。<br/>
　　<br/>
　　弗农姨父则和佩妮姨妈则在叨叨个不停，完全没有制止达力的意思。<br/>
　　<br/>
　　皮尔听他说了一阵之后，问道：“我说，达力，把你的小女仆借我一天怎么样？不过你得给她换一身，瞧她穿的什么，是你的旧衣服吗？”<br/>
　　<br/>
　　一个女孩子，穿着胖子达力的衣服，到处都松松垮垮的，真不入眼。<br/>
　　<br/>
　　达力吃吃地笑着：“明天你可以来我家，给她穿上我妈妈的衣服，然后使唤她一整天。”<br/>
　　<br/>
　　哈莉白了他一眼，鉴于姨父姨妈在场，她没有多说话。<br/>
　　<br/>
　　达力那塞满肥油的脑子里像是想起了什么可怕的事，有些担心地看着哈莉。<br/>
　　<br/>
　　皮尔没有注意这些，还在兴头上巴拉巴拉地说着话。<br/>
　　<br/>
　　来到动物园之后，哈莉仍然是他们的使唤对象，替他们拿着所有买好的纪念品，他们却吝啬得只给哈莉买了一个小小的冰激凌。<br/>
　　<br/>
　　下午，他们来到爬虫馆，皮尔和达力对馆内最大的一只蟒蛇产生了兴趣，趴在玻璃上大喊大叫，想让睡着的蟒蛇醒过来。<br/>
　　<br/>
　　但蟒蛇还是坚持睡着，他们只好无奈地走开了。<br/>
　　<br/>
　　哈莉来到它面前，同情地看着它。<br/>
　　<br/>
　　她感觉它和她一样，被束缚在一个鬼地方，不知道什么时候能够离开。<br/>
　　<br/>
　　这时，大蟒睁开了眼睛，和哈莉对视了一会儿，发出嘶嘶的声音。<br/>
　　<br/>
　　哈莉发现自己居然能听得懂，它在说：“嗨。”<br/>
　　<br/>
　　哈莉笑了：“嗨。”她注意到一旁的牌子上写着“蟒蛇，巴西，这是本动物园内繁殖的样品”，顿时明白这条蟒蛇是在这个动物园里繁殖的，她不禁问道：<br/>
　　<br/>
　　“你从没有离开过这里吗？”<br/>
　　<br/>
　　蟒蛇摇了摇头，眼神里透出悲伤：“不，我是从巴西来的。”<br/>
　　<br/>
　　哈莉看了一眼在旁边聒噪的达力和皮尔两个，贴着玻璃小声道：“我放你出去，你给刚才的两个小孩一点教训怎么样？”<br/>
　　<br/>
　　蟒蛇正要点头，皮尔已经发出一声尖叫：“达力！德思礼先生！你们快过来！”<br/>
　　<br/>
　　达力一个箭步窜了过来，他粗鲁地将哈莉推过一边，没有看到哈莉嘴边的笑意。<br/>
　　<br/>
　　两个男孩把脸贴在玻璃上，忽然两个人一起大叫起来，蟒蛇馆的玻璃全都不见了，他们两人一起栽了进去，刚巧栽在巨蟒身前。<br/>
　　<br/>
　　巨蟒嘶嘶地叫着，从他们身上往外爬去，他们以为它会吃了他们，拼命地大喊救命，吓得裤子都湿透了。<br/>
　　<br/>
　　整个爬虫馆的人都往外跑去，哈莉也跟着往外跑，蟒蛇溜过哈莉的身边，向她道了谢。<br/>
　　<br/>
　　哈莉冲它眨了眨眼：“我应该谢谢你，我想这两个小混蛋可以消停一阵子了。”<br/>
　　<br/>
　　巨蟒笑了起来，消失在门外。<br/>
　　<br/>
　　等混乱过去，所有人都坐在汽车里时，哈莉感到弗农姨父已经快气得冒烟了——这里唯一跟穿着斗篷的那群人有联系的，除了哈莉还有谁呢？<br/>
　　<br/>
　　那玻璃不是哈莉弄掉的还能是谁？不过现在皮尔在车上，他不能让皮尔发现他家里有个怪人，所以他就一直生着闷气。<br/>
　　<br/>
　　上车前，达力一定要皮尔坐在哈莉的身边，而自己却坐在皮尔的身边。<br/>
　　<br/>
　　两个人浑身散发着尿骚味，毕竟谁也不会想到去动物园会尿裤子，所以谁也没有带备用的裤子。<br/>
　　<br/>
　　达力一直发着抖，不敢看哈莉。<br/>
　　<br/>
　　哈莉面无表情，皮尔也没心情第二天再来达力家里使唤她了，不过临下车前，皮尔叮嘱达力说：“达力，让你的小女仆不要在学校把这件事散布出去，不然的话，我们就没脸在学校待下去了，知道吗？”<br/>
　　<br/>
　　弗农从前面转过头来：“放心吧，皮尔，她没机会告密的。”<br/>
　　<br/>
　　哈莉知道，弗农姨父打算给她来一个长期的惩罚了，不过她也不怕，因为她可以用很多方法打发这些时光，不跟达力这些混球在一起，倒是她的解脱。<br/>
　　<br/>
　　不过她敢肯定，在今天受到教训之后，达力会不会还有勇气惹自己。<br/>
　　<br/>
　　下车时，达力跟在哈莉后面，小声问道：<br/>
　　<br/>
　　“哈莉，刚才是你做的吗？”<br/>
　　<br/>
　　“半年前我就警告过你，达力表哥。”哈莉看了他一眼，回答道。<br/>
　　<br/>
　　达力愣了一下，没有说话。<br/>
　　<br/>
　　哈莉头也不回地说了一句：“等一会儿你知道自己要做些什么吗，达力表哥？”<br/>
　　<br/>
　　达力不由自主地点了点头：“知道了，哈莉。”<br/>
　　<br/>
　　说完，她走进门里，任由弗农姨父把自己关在储藏室里，他叫嚣着要饿死“该死的哈莉”。<br/>
　　<br/>
　　晚上，达力带着一大块巧克力蛋糕钻了进去：“哈莉，我希望以后我们能够和平共处。”<br/>
　　<br/>
　　“真高兴你想通了，我亲爱的达力表哥。”深吸了一口巧克力的香味，哈莉露出一个赞许甜笑，接过蛋糕开始吃了起来。<br/>
　　<br/>
　　等暑假过去，哈莉也该去上学了，全家都在庆祝达力进入新的中学，佩妮姨妈还在夸奖达力是这个世界最帅的男孩。<br/>
　　<br/>
　　哈莉想起达力那些朋友，还有小学时期那群欺负过她的男生，心想男孩子真是这个世界上最讨厌的动物，而达力则是里面最最讨厌又最丑的一个。<br/>
　　<br/>
　　真希望自己进入新学校的时候，能交几个温柔的女孩子做朋友。<br/>
　　<br/>
　　不过回头仔细想想，其实女孩子、老师们也没有什么特别值得期待的地方。<br/>
　　<br/>
　　没有多少人愿意透过你破破烂烂的衣物，去认识你的内心的。<br/>
　　<br/>
　　女孩子们都在研究裙子、名牌包、帽子、纽扣什么的，谁也不理会土里土气的、穿着男生衣服的哈莉。<br/>
　　<br/>
　　何况她还有一头永远疯长的头发，好像用多了生发剂一样。<br/>
　　<br/>
　　一天早上，哈莉收到了一封奇特的信，弗农姨父和佩妮姨妈折腾了一阵，不想让她拿到这封信，连达力表哥替她用自己斯梅廷手杖敲打他的父亲，又踢了他母亲，还是没有拿到信，他气喘吁吁地来到储藏室：<br/>
　　<br/>
　　“哈莉，对不起——我没能拿到信——”他吭哧吭哧地说着，脸涨得通红。<br/>
　　<br/>
　　哈莉掐了掐他肥嘟嘟的脸：“我不会怪你的，达力表哥，你尽力了。”<br/>
　　<br/>
　　“谢谢你不怪我，哈莉，我得说——我爸爸他，他从来也没有像现在这样不听我的……”他张了张嘴，还想说些什么，最终从口袋里掏出一块巧克力：“吃点糖果，哈莉，这样心情会好些。”<br/>
　　<br/>
　　“弗农姨父是不是有什么事瞒着我？”哈莉接过巧克力，问道。<br/>
　　<br/>
　　“应该是的，但我不知道那是什么……”达力说。<br/>
　　<br/>
　　“没关系的，达力表哥，你不要这么紧张。不过我想也许你可以替我看看信里的内容？”哈莉说。<br/>
　　<br/>
　　“哈莉，我爸爸拿到你的信，立刻全都烧了，我根本没有机会偷看。不过我想这件要瞒着你的事一定很重要。”达力说。<br/>
　　<br/>
　　“你真是个天才，达力表哥。我也这么想，不过信都烧了，再想它也没有什么意义了，还是回去睡觉吧。”<br/>
　　<br/>
　　达力立刻往外走去，刚出储藏室，弗农姨父就出现在外头：“哈莉，你把你的东西收拾一下——你长大了，该给你换个房间了。”<br/>
　　<br/>
　　达力帮着哈莉把东西搬上去。<br/>
　　<br/>
　　弗农姨父奇怪地看着达力：“达力，我说你……”<br/>
　　<br/>
　　他大概是想起现在要对哈莉好些，就住了嘴。<br/>
　　<br/>
　　第二天一早，哈莉又来了三封信，当然，全都又被弗农姨父截留了，弗农姨父气得把邮箱钉住了，可是接下来的几天，给哈莉的信不停地从门缝里、窗户里、送奶员的鸡蛋下面、烟囱等等不可思议的地方出现，弄得弗农姨父简直发了疯。<br/>
　　<br/>
　　最后他崩溃了：“给你们五分钟，去收拾你们的衣服，我们立刻离开这鬼地方。”</p>
