br/>　　<br/>　　	“我相信，这些画绝对是埃德温的亲笔之作。只有他才能画出来。”黑发少女呆呆地望着天空，带着哭腔喃喃自语，“对于生命，他透支地太多，现在偿还了。”<br/>　　<br/>　　	此后，舆论的风向出现了动摇，接连不断有学者和鉴定专家发表证实画作出自德拉科之手的声明，不断有人开始赞美画作无与伦比的美。<br/>　　<br/>　　	虽然有关这些画的归属还是个津津乐道的话题，但是现在，没有一个人会否认这些画的伟大，它们确确实实开启了一个崭新的时代，标记了一个盛世的到来。<br/>　　<br/>　　	“他很伟大，因为他的身体里藏有两个灵魂。”有人在德拉科·马尔福的传记中这样写道。<br/>　　</p><p> </p><p>　　	7<br/>　　<br/>　　	“我神往遥远的天境。<br/>　　<br/>　　	于是我振翅奋起。<br/>　　<br/>　　	可我只有半边羽翼，<br/>　　<br/>　　	背脊上的连结处拼命撕扯，<br/>　　<br/>　　	痛彻骨髓，鲜血淋漓。<br/>　　<br/>　　	终究无法凌空飞起。 <br/>　　<br/>　　	你哟<br/>　　<br/>　　	带着至高无上的纯净，<br/>　　<br/>　　	看出我那卑劣的残疾。 <br/>　　<br/>　　	伸手将那懦弱的灵魂救起，<br/>　　<br/>　　	用无形的湛蓝舔舐丑陋的瑕弊。<br/>　　<br/>　　	如果暴风和急雨曾经来临， <br/>　　<br/>　　	那么烈火就能平静地泯然沉寂。”</p><p>　　<br/>　　	女孩轻声吟唱，火光映亮了她的脸庞，蔓延到黎明的天际。<br/>　　<br/>　　	黎明的天际是最温柔的灰蓝，满天星只剩几颗稀疏地缀在其中，在即将到来的日光中坚持最初的纯粹。正如那个金发青年眼眸的明澈。当你屏息凝神细细观察，那里面也盛着一团火，幽幽的绿色轻晃，那是零度以下最深沉最冷峻的燃烧，是两种毁灭进入同一永生。</p><p>　　<br/>　　	“我从地狱来，要到天堂去，正路过人间。”</p><p> </p><p> </p><p>——END——<br/>*（0）中的句子选自司汤达的《红与黑》<br/>*（1）中的诗歌选自彼德拉克的《寄往天堂的情书》，句子选自《圣经》<br/>*（5）中的诗歌选自雪莱的《自由》<br/>*（7）中的诗歌为原创:
