    <h3 class="landmark heading" id="work">Chapter Text</h3>
    <p> </p><p>
  <strong class="ql-author-16839465 ql-size-12">1. 阿不思·波特</strong>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“嘿，你听说了吗？马上就要有别的学校的人过来啦。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“是呢，我很好奇那两所学校的学生都是什么样子的。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“听说法国盛产媚娃，不知道布斯巴顿来的人里会不会有……”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">……</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思沉默地听着公共休息室里叽叽喳喳的讨论声，收拾了东西起身准备回寝室。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“嘿，波特。”身后传来一道不怎么友好的声音，“你觉得那些外国访客们会不会认识你呢？他们会不会也马上就发现著名的哈利·波特的孩子是个怪胎呢——”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">周围响起一阵窃笑声。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思的身体僵了僵，但他没有回头，而是径直走出了斯莱特林公共休息室。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">又来了。这种事情为什么总是没完没了？</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">有时候他真想换个身份——当然这是不可能的。谁都可以轻易改换身份，唯独他，不行。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思·西弗勒斯·波特，是巫师届大名鼎鼎的英雄哈利·波特的儿子。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">这注定了从他开始出现在公众视野中起，就一定会被各种有形与无形的镁光灯包围，被所有人注视、评判，并被迫接受一切友好善良和尖酸刻薄的评价。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">他的爸爸对此一直感到很歉疚，但阿不思知道这不是他的错。谁让他爸爸是个大名人呢，这些事情总是无法避免。可懂得这个道理不意味着他不会对此产生厌恶，谁也不会喜欢自己所做的一切都被一群不认识的人牢牢盯着，何况他还只是一个十一岁的小孩。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">更糟糕的是，在阿不思被分院帽分去斯莱特林后，这一切都变本加厉了。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">所有人都觉得“救世主”的儿子应该是个百分之百纯正的格兰芬多，可是瞧啊，他竟然被分进了与之对立的、在上一场战争里变得臭名昭著的学院！</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">一时间所有的舆论都对准了他，所有的学生都用另类的眼光看他，包括同学院的学生——他们中大部分的父母都是上一代的斯莱特林，终究跟波特不是同一个世界的人，而他毕竟是救世主的儿子，自然也不会受待见。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">这还不是全部。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">手拿羽毛笔带着闪光灯的大人总是没完没了地问阿不思关于他妈妈的事情，入学后许多年龄大一些的孩子也时常拿他的家庭状况取乐，甚至有六七年级的学姐跑来说要做他的后妈——</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">没错，阿不思是来自单亲家庭的孩子。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">从他记事起，家里就只有自己和爸爸两个人。他曾问过爸爸，为什么别人家都有爸爸和妈妈两个人？他的爸爸沉默了很久，最终告诉他，妈妈因为一些原因不得不离开自己，但她和爸爸一样永远都爱他。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">六七岁的时候，阿不思从报纸上读到了一则新闻，是关于一对在战争中失散的母女时隔多年终于重聚的故事。由此，他想要寻找妈妈的愿望空前强烈起来。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">那段时间，他疯狂执着于挖掘自己素未谋面的神秘母亲的信息，他的爸爸对此却坚决拒绝多透露一个字，总是以各种各样的方式搪塞他的问题。最终他不得不放弃，可那谜团却一直如同幽灵般横亘在心头，心底对于见到妈妈的渴望从未消失。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思真的很讨厌那些嘲笑他家庭的声音，讨厌高年级学生开的他听不懂但绝对不怀好意的“玩笑”，讨厌一切把他当成怪物看的目光。</span>
</p><p class="ql-long-16839465">
  <strong>
    <span class="ql-author-16839465">他来到霍格沃茨才一个多月，可是一切都已经糟透了。</span>
  </strong>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">至于那些学生今晚的谈话中说到的外国访客们，阿不思完全不关心。他也不知道那些外国人到底为什么要来，无论是法国、德国、西伯利亚的学生还是山里的巨怪要来霍格沃茨干什么，都跟他没有关系。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">要不是图书馆在宵禁后不开放，他都不会选择待在黑湖底下的公共休息室——所以实际上他本来连这件事都不会听说。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">梅林知道他只是想找个安静偏僻的地方自己待着而已。</span>
</p><p> </p><p> </p><p>
  <strong class="ql-author-16839465 ql-size-12">2. 三强争霸赛</strong>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思完全不关心的事，正是时隔十五年再次在霍格沃茨举办的三强争霸赛。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">上一次在这里举办的三强争霸赛几乎是噩梦的开始，在最后一项赛事中，选手塞德里克死亡，哈利·波特则亲眼目睹了伏地魔的复活。那对于许多人来说都不是什么美好的记忆，好在现在一切都已经过去很多年了，起码不需要再担心校园里突然冒出个黑魔头之类的东西。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">这一次的欢迎宴会十分隆重。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">即便再事不关己，第一次见到外来巫师总还是好奇的。阿不思偷偷抬头观察着两个代表团，布斯巴顿的巫师们都穿着衣料上好的淡钴蓝制服，女巫们气质优雅大方；德姆斯特朗的巫师们则身着深红色的外衣，并披着皮裘斗篷。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">不得不承认布斯巴顿的女巫们真的很好看呀。阿不思有点脸红，这就是媚娃的魅力吗？</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">台上的麦格校长开始对这两所学校作一一介绍，讲到三强争霸赛的具体事宜时阿不思又失去了兴趣。既然是年满十七岁才能参加的比赛，那跟他真的是毫无关系了。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">一年级的小巫师无聊地打起了哈欠，对周围兴奋的讨论声毫无兴趣，只盼着能快快回到宿舍去。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">欢迎仪式结束的第一时间，阿不思就趁人不注意溜出了大厅。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">但愿这些外国巫师们能暂时分散一下大家的注意力，他心想，这样自己就可以暂时安宁一阵子了。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思没有立刻回宿舍，他跑去天文塔上待了一会，待到宴会散场、大部分人都回到了休息处后，才掐着宵禁时间匆匆往下赶。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">快要走到地下休息室入口的一个拐角处，低着头猛跑的他撞上了一个人，随着一声痛呼，他狼狈地捂着脑袋跌坐在地上。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">对方似乎抱了一大摞书，此刻全都因为冲撞散落在地上。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“梅林的胡子，我很抱歉——”阿不思的声音在看到对方面孔的那一刻戛然而止。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">他几乎要以为自己其实撞在了一面巨大的镜子上——除了镜子里的这个“自己”，有着一头与他乱糟糟的黑发截然不同的光滑柔顺的金发。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">那正是爸爸给他用染发剂之前，阿不思原本的发色。</span>
</p><p> </p><p> </p><p>
  <strong class="ql-author-16839465 ql-size-12">3. 斯科皮·马尔福</strong>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">斯科皮·马尔福的方向感向来不怎么好。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">考虑到他从小就在宽敞巨大的庄园里长大，这显然不太正常。不过由于他从来没有什么机会单独一人外出，在家里也总有小精灵们为他服务，似乎也就不足为奇了。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">斯科皮在法国南部长大，家住普罗旺斯附近的一个小郡，今年是他在布斯巴顿魔法学校上学的第一年。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">尽管一家都是英国人，但这次跟随学校来到霍格沃茨，却是斯科皮11年以来第一次踏上家乡的土地。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">父亲从未和他讲过当初搬到法国的原因，平时也不怎么会谈英国的事，只是模糊地提起过当年在英国曾经发生过一场对社会影响很深的战争。他猜想那也许就是马尔福一家离开的主要原因。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">初次到访，斯科皮对这里的一切都充满了好奇。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">这里是父亲的母校呢，他边走边看，心中涌起一股奇异的感觉，他的双亲似乎就是在这里认识彼此的。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">斯科皮一直知道自己的家庭和一般人不太一样。家里唯一的女性是他的奶奶，父亲很早就告诉过他，他不是没有妈妈，而是有两个爸爸。关于另一位爸爸，他所知并不多，多数印象只是来自于父亲偶尔的只言片语和几张不怎么清晰的照片。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">霍格沃茨的城堡没有布斯巴顿的宫殿那么雄伟壮丽，但却足够新奇有趣，这里有许多他的学校不会出现的东西，比如无头骑士幽灵和随时会变换方向的楼梯。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">好吧，他得承认自己被后者折腾得够呛，自从和大部队走散后就一直在这迷宫一样的城堡里晕头转向。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">多余的访客校舍不够两个学校代表团里所有的学生住，他们这些低年级的学生被统一安排到了空余的本校学生宿舍里。然而斯科皮自带路痴属性又初来乍到，在楼梯上转了半天都没找到通往地下室的正确道路。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">然后在第7次绕过同一个路口的时候，“砰”的一声，他被人撞了个满怀。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">那人坐在地上捂着头痛呼，斯科皮首先注意到的是对方身上霍格沃茨的校袍，顿时眼睛一亮，总算碰到个可以问路的了。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">接着他就看到了对方和自己几乎一模一样的脸，连同对视的那一刻戛然而止的道歉。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“……”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">那个和自己只有发色不同的男孩愣愣地眨了眨眼，突然转身就跑，速度之快令斯科皮咂舌，想追上去时他早就跑得连影都不剩了。难道我看起来跟鬼一样可怕吗，斯科皮纳闷地想。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">那天更晚些的时候，斯科皮终于在一位老师的带领下找到了自己的宿舍。他的宿舍在黑湖底下，透过窗户就能看到湖里的景象，水生生物们在模糊的月光中影影绰绰，斯科皮几乎看呆了。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">霍格沃茨真是太有趣了，他心想。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">临睡前他又想起了刚刚碰到的男孩。这世界上怎么会有长得这么像的陌生人呢，难道真的存在“另一个我”这种神奇的东西？他迫不及待想再见到那个男孩从而一探究竟了。</span>
</p><p> </p><p class="ql-long-16839465">
  <span class="ql-author-16839465">幸运的是他第二天早上就再次看到了那个黑发的“自己”。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">可当斯科皮上前去打招呼的时候，那男孩却皱着眉头跑开了，似乎完全不想和他有交集。说实话被一张酷似自己的脸拒绝的感觉——有点怪怪的。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">然而斯科皮不打算放弃，他一路跟着黑发男孩来到了大厅，吃完饭又跟着他一起去了位于地下的魔药课教室——反正布斯巴顿其他学生也有不少将跟着同年级的班级上课——并在教室的后面找了个空座位坐了下来。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">两个长相极其相似的男孩在同一个空间里，必定会引起其他人的注意。阿不思在忍受了一整天学生们好奇的、惊讶的、探究的目光，和无数“后面有个人和你长得很像哎”的感叹后，终于忍无可忍。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“你到底想干嘛？”晚上回寝室的路上，黑发男孩突然停在半路，猛地转身看向一直跟在自己后面不远处的金发男孩。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">斯科皮被他吓了一跳，支支吾吾地开口：“我，我没有恶意……”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“不要一直跟着我，行不行？”黑发男孩脸上浮现出一种斯科皮看不懂的痛苦表情，“你跟着我只会给我们招来多余的关注，而我一点也不想要那个。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“噢！抱歉…”他小心翼翼地靠近那个男孩，“但是……我们确实长得很像，你不觉得吗？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“可那又不代表你跟我有什么关系！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“……”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">男孩烦躁地拢了拢身上的袍子。“总之，你别再跟着我了，好吗？不介意的话我现在要走了。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“等一下——”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">黑发男生转过头不耐烦地看着他。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“……你知道斯莱特林的地窖怎么走吗？”斯科皮怯怯地问，“对不起，我好像又忘记路了。”</span>
</p><p> </p><p> </p><p>
  <strong class="ql-author-16839465 ql-size-12">4. 两个爸爸</strong>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“我叫斯科皮，介意告诉我你的名字吗？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“……阿不思。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“你今年几岁呀？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“11岁。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“好巧，我也是11岁。你是哪天生日呀？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思停下了脚步，抿起嘴看向身边一直问个不停的金发男孩。“你到底想干嘛？梅林在上，我跟你真的没有任何关系好吗！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">斯科皮的注意力此时却被阿不思身后墙上的一幅照片吸引了，他的眼睛瞪得大大的，目光越过黑发男孩落在相框里笑着的黑发碧眼少年：“嘿，我见过那个人！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思转头，果不其然看到了自己爸爸和他的两个朋友读书时的合照。他叹了口气，“我想那是桃金娘挂在那里的。”除了她还有谁会把朋友的照片挂在厕所门口呢。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">并不想暴露自己的身份引来新一波的惊呼，他主动拉扯着金发男孩的袖子想把他拉走。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“可是，可是……”斯科皮挣扎着不愿离去，思索过后终于想起到底在哪见过他了：“那个人是我爸爸耶！”</span>
</p><p class="ql-long-16839465"><span class="ql-author-16839465">“什么？！”阿不思皱着眉头瞪他，“你在开玩笑吧？所有人都知道哈利•波特是</span> <strong class="ql-author-16839465"> <em>我的</em> </strong> <span class="ql-author-16839465">爸爸！”</span></p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“不，真的，我没骗你！父亲床头相框里的那个人和他长得一模一样，而且他说过我没有妈妈，但有两个爸爸。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“……Excuse me？！梅林，这不可能是真的！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“哦，这当然是真的。”斯科皮把手伸进校服口袋里掏了掏，“你看，这是我偷偷用复制咒拷贝下来的，你看看是不是和照片上那人长得一样？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思将信将疑地接过照片。画面里一个黑发的少年睡眼惺忪地靠在另一个有着铂金色头发的少年胸前，片刻后突然清醒过来，大笑着抄起枕头往金发少年挂着得意笑容的脸上拍去。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">那个黑发少年，和他曾在杂志报纸上见过的爸爸年轻时的模样分毫不差。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">这么多年来，他也从未见过爸爸露出过那样的笑容，那样纯粹张扬而又充满幸福的活力，是他没有见过的爸爸在那个年纪最原本的少年气。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思知道许多他这个年龄的人本还不能理解的东西。他的爸爸是少年英雄，年纪轻轻便被迫背上了打败黑魔王的重压，战争来临前的那段日子本应是青春期最肆意的岁月，他却只能在愈加紧迫的局势下一天比一天心事重重、沉默寡言，就连和最好的朋友的合照都不再有那么灿烂的笑容。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“梅林……”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">这张照片上的另一个人，曾经一定和爸爸关系很不一般吧……</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“怎么样？我没有骗你吧。”斯科皮取回了照片，对满脸震惊的阿不思摆出一个鬼脸。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思抬头看了他一眼，看起来完全失去了言语的能力。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">斯科皮冲他眨眨眼。“所以，你的生日是哪天？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“……8月19日。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“好极了！1998年8月19日，对吧？我也是那天出生的！”金发男孩兴奋地跳了起来，“梅林，我竟然有个双胞胎兄弟！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“等等等等，你慢着。”阿不思总算回过神来，尽管依然满脸不可置信：“你是在说我们是两个男人的孩子吗？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“是呀。绝不会错的，这是父亲亲口告诉我的。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“这怎么可能？我是说，我爸爸确实告诉过我双亲中的另一位因为某些原因不得不离开，可他说的是‘妈妈’呀？”黑发男孩的眉头深深皱起。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“唔，我想那只是爸爸不想让你因为真相烦恼。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“……可是你确定两个男人真的可以孕育孩子吗？”阿不思往后缩了缩，听到另一个人和他分享“爸爸”的称呼感觉新奇又怪异。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“嘿，我们可是生活在魔法世界，有什么是不可能的？”斯科皮无所谓地摆摆手，“住我家附近的小山姆就是两个叔叔的孩子，他们一家可幸福了呢。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思眨眨眼，似乎还在艰难地消化着刚刚得到的信息。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">长得几乎一模一样的同一天生日的男孩，爸爸年轻时和另一个人的亲密合照，以及那个人和自己原本发色如出一辙的金色头发——似乎忽然间爸爸的一切奇怪行为都解释得通了。</span>
</p><p class="ql-long-16839465">
  <strong>
    <span class="ql-author-16839465">Oh bloody hell......原来我有两个爸爸。</span>
  </strong>
</p><p> </p><p> </p><p>
  <strong class="ql-author-16839465 ql-size-12">5. 天才计划</strong>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“所以你的，不，我们的父亲，他是个怎样的人？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思直接把斯科皮带进了自己的寝室，麻利地给床铺周围丢了个刚学的静音咒，一躺上床就开始问东问西。他实在有太多想知道的东西了，一旦接受了“两个爸爸”的设定，剩下的便是无穷无尽的好奇。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“唔……大家都说父亲很宠我，但他有时候也很严厉，我犯错的时候会被训得很惨。”斯科皮歪了歪脑袋，“我们的爸爸呢？他对你怎么样。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“他对我挺好的，以前经常带我出去玩，也不怎么批评我。但是……他真的很忙啦，有时候就会不能陪在我身边，那时候我就会去赫敏阿姨和罗恩叔叔家住一阵子。”阿不思惆怅地叹了口气，“其实爸爸是大名人的感觉有时候并不怎么好。就像是，你必须要和一大堆人分享他，而那些人还希望能从我身上挖出点什么——这真的很烦人。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">斯科皮撇撇嘴表示赞同：“父亲在我们那边不算是大名人，可还是有好多人想把他抢走，尤其是打扮得花枝招展的阿姨和姐姐们。她们还来讨好我呢，但我一点都不喜欢她们。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“噢，我也是！”他们对视，在对方眼里看到了同病相怜的无奈。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“等等，”阿不思撑起一只手肘侧头看向斯科皮，“我们的金头发父亲，他现在结婚了吗？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“当然没有！悄悄告诉你，他喝醉酒后会一直念叨爸爸的名字。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“爸爸也是，除了赫敏阿姨外，我都没见他和别的女人走得特别近过。”阿不思回想起曾经不能理解的一些事情，“我还偷听到过他跟赫敏阿姨说他不会再有真正的爱情了，因为……因为他人生中唯一的爱情已经——”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“——离他而去了？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“嘿！你怎么知道！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">斯科皮冲他得意地挑挑眉。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“好吧。”阿不思把头枕回了手臂上，“反正这大概就是在说他自己和你爸了。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">他又翻了个身，闷闷的声音从棉被里传出来：“我其实一直挺羡慕有完整家庭的孩子的……哎，好想见见另一个父亲啊。虽然从妈妈变成了第二个爸爸，但其实也没什么差别。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“我也是……诶，等等，等等！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">斯科皮脑海中灵光一现，而后惊喜地一下抓住阿不思使劲摇晃。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“兄弟，你绝对想不到我有了怎样一个天才主意！”他兴奋地就快要从床上蹦起来了，“我们互换吧！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“什么？”黑发男孩探出脑袋，一头雾水。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“圣诞假的时候大家都要回家，我们何不就这个机会互相交换一下呢？你代替我去法国当斯科皮，我留在这儿扮演阿不思——完美极了，不是吗？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思脸上渐渐现出恍然大悟的神色。“梅林…这真是妙极了。可是你确定我们不会穿帮吗？我是说，我们长得并不完全一样，而且性格什么的也很不同。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“这正是为什么我们必须好好利用剩下两个月时间！”斯科皮一拍手，“而且，等他们发现这事之后，迟早会把我们换回来，那就意味着——”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“他们就必须得见面，说不定有些事也就解决了——”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“然后我们就能同时和两个爸爸生活在一起了！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“梅林的袜子！我都没想到还有这一层。这实在是，实在是太聪明了！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“对吧！相信我，这个天才计划是一定会成功的！”</span>
</p><p> </p><p class="ql-long-16839465">
  <span class="ql-author-16839465">七周后。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“是时候检验一下训练成果了。”两个男孩面对面盘腿而坐。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“我很讨厌但是家里每天都会有的菜？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“胡萝卜。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“父亲对麻瓜的态度？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“没以前那么反感但依然不愿接近。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“罗恩叔叔？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“和爸爸一起经常被赫敏阿姨教育，孩子名叫罗丝和雨果。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“马尔福家的规矩？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“保持优雅，适当傲慢。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“我爸最反感的称呼？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“救世主。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“我爸最讨厌的动物？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“雪貂。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“我爸的眼镜多久洗一次？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“从不。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“Perfect！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“等等，”阿不思的脸上突然现出惶恐的神色，“我一点法语都不会讲，这可怎么办？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“哦没事，假期里一般不会出门的，我们在家都用英语交流的！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“对了，差点忘了最重要的事！”斯科皮看着对方的黑发，拍着脑袋跳起来，“我们的头发怎么办？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思略一思索，拉着人就往楼上跑。“这好办，有求必应屋听说过吗？”</span>
</p><p> </p><p class="ql-long-16839465">
  <span class="ql-author-16839465">圣诞假期前夕，布斯巴顿魔法学校的学生即将启程返回法国南部。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">阿不思和斯科皮在门口告别，后者给了前者一个拥抱。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“准备好了吗，兄弟？”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“我不知道，我现在紧张得要死。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“放松Bro，我们会搞定一切的。”黑发男孩贴着金发男孩的耳朵说，“记得保持联系！”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“我们的猫头鹰要很辛苦了。”阿不思笑了起来，“代我照顾好爸爸。”</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“你也是，替我向父亲问好。”斯科皮捏了捏他的手。</span>
</p><p class="ql-long-16839465">
  <span class="ql-author-16839465">“愿梅林赐予我们好运！”</span>
</p><p> </p>
