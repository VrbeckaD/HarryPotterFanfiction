    <h3 class="landmark heading" id="work">Chapter Text</h3>
    <p class="p1" align="center"><span class="s1"><br/></span>宁静的小镇因为这些军官的到来而变得吵吵闹闹，他们在这个镇子上短暂的休憩，过个几天的放浪生活，就得再回到那令人精疲力竭的战场上。他们得把每天都当作最后一天来过，因为下一刻就可能会死在战壕里。</p><p class="p3"> </p><p class="p1">
  <span class="s1">忒修斯和他的几个战友喝得醉醺醺的朝临时的住处走，他们途径一个僻静的巷子，几个妓女在冲他们抛媚眼。</span>
</p><p class="p1"><span class="s2">“</span> <span class="s1">老天，瞧那个姑娘的衣服有多低，快要掉下来了。</span> <span class="s2">”</span> <span class="s1">士兵冲着那些姑娘吹了个下流的口哨。</span></p><p class="p1">
  <span class="s1">那些姑娘笑的花枝乱颤，直邀他们进来快活。</span>
</p><p class="p1">
  <span class="s1">他们已经连续几天经过这个巷子了，总有人想进去玩，但也总有人不同意，他们之中有几个家境好的思想传统的，碍于名声不太愿意花钱买一夜春宵，忒修斯就算一个。</span>
</p><p class="p1">
  <span class="s1">事实上忒修斯英俊又绅士，他来到法国的这段日子，已经有不少姑娘被他迷住，想和他来一场浪漫的异国情缘。在这个战争年代，姑娘们最仰慕穿着制服的男人，即使是妓女，也有愿意不收钱和军官睡觉的。</span>
</p><p class="p1">
  <span class="s1">不是所有军人都像忒修斯这样受欢迎，但男人都有这种需求，花点钱买也值得。</span>
</p><p class="p1">
  <span class="s1">那些妓女们走出来，一人一个把这些军官拉进来，其他人没什么拒绝就走进了这家妓馆，忒修斯只好也跟着进去。</span>
</p><p class="p1">
  <span class="s1">似乎是挣脱了门外的礼仪和道德的束缚，这些男人们被妓女稍微一撩拨就缴械投降了，东倒西歪的就被姑娘们扶进了小房间，阵阵嬉笑声混在屋子里，只剩下忒修斯一个人留在了昏暗的客厅里。他拒绝了来和他搭讪的姑娘，打算一个人先回住处。</span>
</p><p class="p1">
  <span class="s1">显然老鸨并不想放弃任何一个顾客，她揽住了忒修斯，从楼上又喊了个姑娘下来。</span>
</p><p class="p1">
  <span class="s1">她和别的姑娘不太一样，个子很高，她穿着高跟鞋看起来竟和忒修斯没差多少，她穿一件深色的裙子，头发是姜红色的，刘海有些挡住了左眼，看起来很羞涩。</span>
</p><p class="p1">
  <span class="s1">忒修斯听到她和老鸨用法语小声说了些什么，但他的法语不是特别好并没有听清楚。</span>
</p><p class="p1">
  <span class="s1">高个子的姑娘引着他上了二楼，忒修斯实在是想找个地方睡觉就没再拒绝她，二楼看起来能安静一些。</span>
</p><p class="p3"> </p><p class="p1"><span class="s2">“</span> <span class="s1">先生</span> <span class="s2">……</span> <span class="s1">如果您需要的话</span> <span class="s2">……</span> <span class="s1">我可以用嘴</span> <span class="s2">……”</span></p><p class="p1">
  <span class="s1">作为一个妓女，忒修斯觉得她实在算不上专业，拘谨的样子就好像忒修斯是来强奸她的。</span>
</p><p class="p1">
  <span class="s1">然而忒修斯此时并不是很有兴致。</span>
</p><p class="p1"><span class="s2">“</span> <span class="s1">这是你的房间吗？</span> <span class="s2">”</span></p><p class="p1"><span class="s2">“</span> <span class="s1">是的</span> <span class="s2">”</span> <span class="s1">尽管回答了问题，她依旧盯着别处。</span></p><p class="p1"><span class="s2">“</span> <span class="s1">你有多余的毯子吗？</span> <span class="s2">”</span> <span class="s1">忒修斯一边脱衣服一边问道。</span></p><p class="p1"><span class="s2">“</span> <span class="s1">有</span> <span class="s2">”</span> <span class="s1">她打开柜子摸出一条毯子，抱在怀里。</span></p><p class="p1"><span class="s1">此时忒修斯正在解腰带，有些好笑的看着对方站在离自己半米远的地方</span> <span class="s2">“</span> <span class="s1">你不打算把它给我吗？</span> <span class="s2">”</span></p><p class="p1">
  <span class="s1">对方缓步朝他走过来，忒修斯拿过毯子道了谢。</span>
</p><p class="p1"><span class="s2">“</span> <span class="s1">好心的姑娘，你要是能再给我倒杯水就更好了，我实在是口渴。</span> <span class="s2">”</span></p><p class="p1">
  <span class="s1">忒修斯没有猜错，楼上果然要安静的多，等那位红头发姑娘端着杯水回来时，他几乎就快睡着了。</span>
</p><p class="p1">
  <span class="s1">对方进来看到床上的忒修斯显然愣了一下，她把水放到床头的柜子上，盯着忒修斯看了半天才走到另一侧的床边。</span>
</p><p class="p1">
  <span class="s1">忒修斯感觉到对方在看他了，他甚至还在进入睡眠的前一秒闻到对方身上的香味。</span>
</p><p class="p3"> </p><p class="p1">
  <span class="s1">忒修斯久违的睡了一个好觉。</span>
</p><p class="p1">
  <span class="s1">他一睁开眼睛就发现另一双漂亮的绿眼睛正盯着他看，但眼睛的主人马上就受惊的闭上了它们。</span>
</p><p class="p1"><span class="s2">“</span> <span class="s1">早上好。</span> <span class="s2">”</span></p><p class="p1"><span class="s2">“……</span> <span class="s1">早上好</span> <span class="s2">”</span> <span class="s1">绿眼睛的主人是在忒修斯起身开始穿衣服的时候才回复了这一句。</span></p><p class="p1"><span class="s2">“</span> <span class="s1">你叫什么名字？</span> <span class="s2">”</span> <span class="s1">系好衬衫的最后一颗扣子，忒修斯问道。</span></p><p class="p1"><span class="s2">“</span> <span class="s1">莉莉。</span> <span class="s2">”</span></p><p class="p1"><span class="s2">“</span> <span class="s1">很适合你</span> <span class="s2">”</span> <span class="s1">忒修斯端起床头柜上的水，并发现了压在下面的诗集，</span> <span class="s2">“</span> <span class="s1">这是本不错的诗集，莉莉，很遗憾我没时间和你讨论一下了。</span> <span class="s2">”</span></p><p class="p1"><span class="s1">莉莉此时坐在床上，头发有些乱，</span> <span class="s2">“</span> <span class="s1">我</span> <span class="s2">……</span> <span class="s1">我还在学</span> <span class="s2">……”</span></p><p class="p1"><span class="s2">“</span> <span class="s1">那就祝你早日成为诗人，莉莉小姐</span> <span class="s2">”</span> <span class="s1">忒修斯喝掉了那杯水，放下了她的诗集，</span> <span class="s2">“</span> <span class="s1">感谢你的一夜春宵，我睡的很好。</span> <span class="s2">”</span> <span class="s1">他朝莉莉碰了碰帽檐，离开了屋子。</span></p><p class="p1">
  <span class="s1">明明两人什么都没有做，莉莉仍然脸红了。</span>
</p><p class="p3"> </p><p class="p1"><span class="s1">忒修斯不打算等那些呼呼大睡的战友们直接离开，早晨七点钟外面还很安静。莉莉</span> <span class="s2">“</span> <span class="s1">噔噔噔</span> <span class="s2">”</span> <span class="s1">地跑下楼，她甚至没有穿鞋子。</span></p><p class="p1"><span class="s2">“</span> <span class="s1">先生</span> <span class="s2">……</span> <span class="s1">我</span> <span class="s2">……</span> <span class="s1">我</span> <span class="s2">……”</span> <span class="s1">她似乎在组织语言，忒修斯看到了她手里拿着诗集，握住她拿着诗集的手，抢在她之前开口。</span></p><p class="p1"><span class="s2">“</span> <span class="s1">帮我个忙莉莉，告诉其他人我们该做的都做了，毕竟你也不想被人认为没有魅力吧。</span> <span class="s2">”</span></p><p class="p1"><span class="s2">“</span> <span class="s1">可是</span> <span class="s2">……”</span> <span class="s1">莉莉仍然犹豫了，因为她的诗集里夹着比她平时多一倍的钱，</span> <span class="s2">“</span> <span class="s1">我总得为你做点什么。</span> <span class="s2">”</span></p><p class="p1">
  <span class="s1">忒修斯看着他面前的这个姑娘，内向又单纯，或许是表象，但总让他有一丝怜悯，觉得她不该是个妓女。</span>
</p><p class="p1"><span class="s2">“</span> <span class="s1">那就替我祈祷吧，活过这场战争。</span> <span class="s2">”</span></p>
